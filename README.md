# 🏏 IPL Analytics Dashboard

## 📌 Project Overview

The IPL Analytics Dashboard is an interactive Power BI project designed to analyze Indian Premier League (IPL) data and provide insights into team performance, player statistics, match outcomes, and season-wise trends. The dashboard transforms raw IPL datasets into meaningful visualizations using Power BI, DAX, and data modeling techniques.

This project enables users to explore IPL performance metrics through dynamic filters, KPIs, and interactive reports.

---

## 🎯 Objectives

- Analyze IPL team performance across seasons.
- Evaluate player contributions in batting and bowling.
- Identify top-performing teams and players.
- Compare match outcomes and toss decisions.
- Provide season-wise IPL insights through interactive visualizations.

## 📂 Dataset Information

The dashboard is built using IPL datasets containing:

- Match-level information
- Ball-by-ball records
- Team details
- Player information

### Data Tables

| Table Name | Description |
|------------|------------|
| IPL Matches Data | Match results, season details, venues, toss information |
| Ball by Ball Data | Delivery-wise match records |
| Players Data | Player information and statistics |
| Teams Data | IPL team details |
